¿Qué es DOCKER?

Docker es una plataforma de software que le permite crear, probar e implementar aplicaciones
			de manera rápida.
		
La idea para la que fue creado docker es la de crear contenedores portables y ligeros. 
		   Estos contenedores se podran ejecutar en cualquier máquina linux que tenga instalado Docker.
		   Para poder ejecutar estos contenedores en otro sistema operativo que no sea Linux, este sistema operativo
		   ya sea windows por ejemplo, tendrá que tener una máquina virtual de linux para poder ejecutar contenedores 
		   docker. Los contenedores docker son más eficientes, con respecto a la memoria, que las máquinas virtuales,
		   ya que comparten el mismo kernel el sistema Linux donde se ejecuta
